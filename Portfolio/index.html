Sudoku Solver

Android

Sudoku Solver

This in one of my Android mini projects for my Mobile Application Development Laboratory that helped me to get a grip on the new tech in under a month. The initial plan was to make an OCR App that can read the Sudoku from the camera and display the answer. But half way through it proved to be very challenging, one of the reasons being that I only had and emulator on windows with no webcam. So I was not able to include the OCR. Instead, I created an app that can solve a sudoku problem when the user enters the problem manually.
